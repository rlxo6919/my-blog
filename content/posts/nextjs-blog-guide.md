## Next.js 블로그

Next.js는 React 기반의 풀스택 프레임워크입니다.

### 왜 Next.js인가?

1. **서버 사이드 렌더링** — SEO에 유리합니다
2. **정적 생성** — 빠른 성능을 제공합니다
3. **파일 기반 라우팅** — 직관적인 구조입니다

### 시작하기

```bash
npx create-next-app@latest my-blog
```

이 명령어 하나로 프로젝트를 시작할 수 있습니다.
